== Description ==

A very basic SEO plugin meant to handle essential SEO functions only. For social meta data (e.g. Open Graph tags and/or Twitter cards) please install Jetpack by wordpress.com.

== Installation ==

1. Upload "essential-seo" to the "/wp-content/plugins/" directory
2. Activate the plugin through the 'Plugins' menu in WordPress
